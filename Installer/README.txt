BabyShop installer
=================

Application files are installed for the current user into:
%LOCALAPPDATA%\BabyShop\app

Important:
- The application expects MySQL / MariaDB to be available locally.
- The configured database name is baby_shop_restored.
- If the database is not restored yet, import the current SQL dump before first use.

Uninstall:
- Run %LOCALAPPDATA%\BabyShop\uninstall.cmd
